🌤️ Météo HA - Glassmorphism DashboardUne interface météo interactive et esthétique conçue pour Home Assistant, utilisant le design Glassmorphism. Cette page affiche en temps réel les conditions climatiques, les prévisions, les niveaux de pollen et les vigilances, le tout avec un fond animé synchronisé sur la météo réelle.✨ CaractéristiquesDesign Glassmorphism : Interface moderne avec effets de transparence, flous et bordures lumineuses.Environnement Dynamique : Fond animé en Canvas (Soleil, pluie, orage, neige, brouillard, jour/nuit) basé sur l'état de vos capteurs.Suivi complet du Pollen : Affichage détaillé par type (Graminées, Bouleau, etc.) avec barres de progression colorées.Vigilance Intelligente : Système d'alertes automatiques (Canicule, vent fort, orages).Mode Démo : Barre d'outils intégrée pour tester chaque scène visuelle sans attendre le changement de météo.Configuration Facile : Interface de réglage intégrée pour lier vos entités Home Assistant sans modifier le code.📸 Aperçu des Scènes☀️ Soleil🌧️ Pluie⛈️ Orage🌙 NuitRayons et particulesGouttes animéesÉclairs (Flash)Étoiles et Lune🚀 Installation rapideHébergement : Déposez le fichier index.html sur votre serveur (ou dans le dossier /config/www/ de votre instance Home Assistant).Accès : Ouvrez la page dans votre navigateur ou via une carte Webpage sur votre tableau de bord Lovelace.Connexion :Au premier lancement, la carte vous demandera un Jeton d'accès longue durée (Long-Lived Access Token).Générez ce jeton dans Home Assistant (Profil utilisateur -> Tout en bas).Configuration : Cliquez sur l'icône ⚙️ Config pour renseigner les noms de vos entités (ex: weather.ma_maison, sensor.pollen_gramines, etc.).🛠️ Entités supportéesPour un affichage complet, il est recommandé d'utiliser les intégrations suivantes dans HA :Weather (Pirate Weather, Météo-France, AccuWeather).Pollen (Intégration "Pollens" de France).Qualité de l'air (IQVIA ou capteurs locaux).Soleil (Entité sun.sun par défaut).🗂️ Structure du codeCSS : Gestion du flou (backdrop-filter) et des bordures dynamiques selon la scène.Canvas API : Moteur de rendu haute performance pour les particules (pluie, neige, étoiles).WebSocket : Communication bidirectionnelle en temps réel avec l'API de Home Assistant.LocalStorage : Sauvegarde locale de vos paramètres d'entités pour ne pas avoir à les ressaisir.📝 LicenceCe projet est sous licence MIT.
